Getting Started

First you’ll want to click on the Download button to open the application in a new browser window. Depending on what device you’ve opened the list application on and/or in, below will instruct you on how to access your lists quickly!

Desktop:
If you’ve clicked the Download button and opened the application in a new browser window, all you need to do is Bookmark the page, and make it a favorite.

Tablet and/or Phone:
Once you click the Download button to open the application in a new browser window, you’ll want to add the list “page” to your home screen. All devices and manufactures have this feature, however, not all are location in the same place.

Using Symplist

How do I create a To-do?
On iPhone, iPad, Android, as well as, Mac and Window desktop’s, tap or click the “+” button in the bottom right hand corner. This will provide you with a prompt to enter your item.

How do I remove/delete an To-do?
Just tap or click on the list item you want to remove viola, it’s complete!

Q: How secure is my data with Symplist?
A: Symplist currently only utilizes your devices’ localStorage capabilities and you information is not stored, shared, backed up and/or hosted on any databases or web servers by Symplist App.

A: Unfortunately, not at this time.

Q: Which web browsers does Symplist support?
A: Symplist works with all modern web browsers and recent versions of Chrome, Safari, Firefox, and Internet Explorer.
